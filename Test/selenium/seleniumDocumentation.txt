To run the .side automated test file use the Selenium IDE by Selenium.
This is an extension you can add on to your firefox or Chrome browser from
their respective web stores.
Selenium IDE is an integrated development environment for Selenium tests.
It is implemented as a Firefox extension, and allows you to record, edit, and debug tests.
